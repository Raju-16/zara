# ZARA Clone
It is a clone of Zara.com

Check the LIVE hosted website [here](https://zara-vert.vercel.app/) or check screenshots below.
## Features

- Sign-in/Sign-up
- Landing, Product, and, Help pages
- Filter acc. to price.
- Dynamic shopping cart
